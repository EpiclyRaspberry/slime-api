Welcome to the API!

This is an introduction to the API and its functionality.

You can use the API to perform specific tasks or retrieve data.

Example Usage

    
fetch('/slime', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    tables: [
      [chunkX1, chunkZ1],
      [chunkX2, chunkZ2],
      [chunkX3, chunkZ3]
    ]
  })
})
  .then(response => response.json())
  .then(data => console.log(data));
